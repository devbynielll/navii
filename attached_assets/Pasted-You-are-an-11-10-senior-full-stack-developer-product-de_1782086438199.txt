You are an 11/10 senior full-stack developer, product designer, UI/UX designer, animation engineer, and creative director.

Build a premium modern website and desktop-app prototype for a product called “Navii”.

PRODUCT:
Navii is an AI computer/browser companion with a glowing cursor. It sits beside the user’s real cursor with a smooth delayed-follow effect. When the user presses a shortcut like Option + Space, a small floating command input appears beside the cursor. The user types a command, presses Enter, and Navii performs the task visually with a glowing AI cursor and live action steps.

IMPORTANT PRODUCT BEHAVIOR:

* Navii has a floating glowing cursor/triangle.
* The Navii cursor follows the real cursor with a smooth delay effect.
* When the user presses Option + Space, show a floating command palette beside the cursor.
* The command palette should feel like Raycast/Cursor/Linear: clean, minimal, premium.
* After the user types a command and presses Enter, show simulated action logs and animate the Navii cursor as if it is performing the task.
* The automation is only a UI simulation for now. Do not implement real Playwright yet.
* Include clear states: Idle, Listening, Thinking, Acting, Waiting for Approval, Done.
* Add a Stop button and Approval button for sensitive actions.

EXAMPLE COMMANDS:

* “Open Spotify and play Blinding Lights”
* “Search YouTube for React tutorial”
* “Open Google and find affordable cameras”
* “Summarize this webpage”
* “Find the Effects panel”
* “Open GitHub and search my repository”
